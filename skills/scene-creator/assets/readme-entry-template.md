# README 索引条目参考

在对应模块的 README.md 中添加新场景时，参考以下格式。

---

## acceleration 模块风格

`references/acceleration/README.md` 使用三列表格：文件、风险等级、触发场景。

追加一行示例：

```markdown
| [<scene-name>.md](<scene-name>.md) | <风险等级> | 「<触发短语 1>」「<触发短语 2>」「<触发短语 3>」 |
```

风险等级参考：
- 纯读操作 → 低风险
- 涉及写操作但影响可控 → 中风险
- 涉及高影响写操作（创建/删除站点等） → 中高风险

---

## security 模块风格

`references/security/README.md` 使用三列表格 + 快速决策树。

### 表格条目

```markdown
| [<scene-name>.md](<scene-name>.md) | 🟢 低风险 | <核心触发场景一句话描述> |
```

风险 emoji：
- 🟢 低风险（纯读 / 查询）
- 🟡 中风险（涉及写操作但影响可控）
- 🔴 高风险（涉及高影响写操作）

### 决策树条目

在 `## 快速决策树` 的代码块中追加：

```
├─ 「<触发短语 1>」
│  「<触发短语 2>」
│  └─ → <scene-name>.md  <风险emoji> <风险等级> · <一句话说明>
```

---

## 新模块风格

如果创建了全新的模块目录，README.md 可参考 acceleration 或 security 的格式，选择更适合的即可。建议包含：

1. 一级标题和一句话说明
2. `## 本目录文件` 表格
3. 可选：快速决策树、前置条件、参考链接
